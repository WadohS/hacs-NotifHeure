# NotifHeure - Intégration Home Assistant

Intégration **notify** pour les panneaux [NotifHeureXL de Byfeel](https://byfeel.info/notifheurexl/).

## Installation via HACS

1. Ajouter ce repo HACS : `https://github.com/WadohS/hacs-NotifHeure`
2. Rechercher "NotifHeure" → Installer
3. Redémarrer Home Assistant

## Configuration

**Intégrations** → **Ajouter** → **NotifHeure** → Ajouter tes panneaux :
- Nom : `bureau`
- Topic : `byfeel/NotifheureXL/f52d9/bureau/message`
